MAE (Mean Absolute Error)

What is MAE?

Mean Absolute Error (MAE) is a metric used to measure the average magnitude of errors between predicted and actual values in a dataset, without considering their direction (positive or negative).

Key Characteristics:

  • Robust to outliers - Less sensitive than MSE/RMSE
  • Easy to interpret - Same units as the original data
  • Always positive - Uses absolute values
  • Linear scoring - All errors are weighted equally

MAE Formula

MAE = (1/n) Σ |yj - ŷj|

Formula Components:

  • n = Number of observations/data points
  • yj = Actual value for observation j
  • ŷj = Predicted value for observation j
  • |yj - ŷj| = Absolute difference (residual)
  • Σ = Sum of all absolute differences

Steps to Calculate MAE:

  1. Calculate the residual (error) for each data point
  2. Take the absolute value of each residual
  3. Sum all absolute residuals
  4. Divide by the number of observations

RMSE Formula Deep Dive

RMSE = √[(1/n) × Σ(yi - ŷi)²]

RMSE Calculation Steps:

  1. Calculate residuals: (yi - ŷi) for each observation
  2. Square each residual: (yi - ŷi)² - eliminates negatives and amplifies large errors
  3. Calculate mean: Sum all squared residuals ÷ n = Mean Squared Error (MSE)
  4. Take square root: √MSE = RMSE (returns to original units)

Why RMSE Differs from MAE:

  • Squaring amplifies large errors: A error of 2.0 becomes 4.0 when squared
  • Square root moderates the effect: But large errors still have disproportionate impact
  • More sensitive to outliers: RMSE increases more than MAE when outliers are present
  • Geometric vs Arithmetic mean: RMSE is like a geometric mean of errors
Error Value Absolute Value Squared Value Impact Factor
0.2 0.2 0.04 Small errors minimized
0.5 0.5 0.25 Moderate amplification
2.0 2.0 4.0 Large errors amplified 2x

MAE vs RMSE: Detailed Comparison

Scenario MAE Calculation RMSE Calculation MAE Result RMSE Result Ratio (RMSE/MAE)
Scenario 1 (0.4+0.2+0.3+0.5+0.2)/5 √[(0.16+0.04+0.09+0.25+0.04)/5] 0.32 0.34 1.06
Scenario 2 (0.4+0.2+0.3+0.5+2.0)/5 √[(0.16+0.04+0.09+0.25+4.0)/5] 0.68 0.95 1.40

Key Insights from the Comparison:

  • Ratio = 1.00 (Scenario 3): Perfect uniform error distribution
  • Ratio ≈ 1.06 (Scenario 1): Relatively consistent errors with small variation
  • Ratio = 1.40 (Scenario 2): Significant outlier present - RMSE penalizes heavily
  • Rule of thumb: When RMSE/MAE > 1.2, you likely have outliers

The Mathematics Behind the Difference:

In Scenario 2, the outlier error of 2.0:

  • Contributes 2.0/3.4 = 58.8% to the total absolute error sum
  • Contributes 4.0/4.54 = 88.1% to the total squared error sum
  • This is why RMSE (0.95) is much larger than MAE (0.68) for this scenario

When to Use MAE vs RMSE?

Use MAE When:

  • Robust evaluation needed: Outliers shouldn't dominate the metric
  • Linear cost of errors: All prediction errors have similar business impact
  • Easy interpretation required: MAE is in original data units
  • Median-focused models: MAE optimizes for median prediction

Use RMSE When:

  • Large errors are costly: Want to heavily penalize big mistakes
  • Gaussian error assumptions: Errors are normally distributed
  • Standard in your field: Common in regression analysis
  • Mean-focused models: RMSE optimizes for mean prediction

Summary & Best Practices

When to Use MAE:

  • Robust evaluation needed: When outliers shouldn't dominate the metric
  • Interpretability important: MAE is in the same units as your data
  • Linear penalty preferred: All errors treated equally
  • Business context: When all prediction errors have similar cost

Advantages of MAE:

  • Easy to understand and interpret
  • Robust to outliers
  • Same units as original data
  • Linear relationship with errors

Quick Reference Formula:

MAE = (1/n) × Σ |actual - predicted|

Remember: Calculate absolute differences, sum them up, then divide by the number of observations!
